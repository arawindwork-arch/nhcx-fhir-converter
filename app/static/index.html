Active Security Controls

🔐

AES-256-GCM Encryption

All extracted PHI/PII data encrypted at rest. Keys managed via environment variables (production: AWS KMS / Azure Key Vault).

📋

Immutable Audit Trail

Every data access, conversion, and download logged with who/what/when/where. Tamper-detected via SHA-256 integrity hashes.

🔑

API Key Authentication

X-API-Key header authentication with 4 role levels: admin, reviewer, api, readonly. Demo mode available for hackathon testing.

⏱️

Rate Limiting

30 requests/minute per client. Token-bucket algorithm prevents abuse while allowing burst traffic for legitimate use.

🗑️

Secure Data Deletion

DoD 5220.22-M secure deletion — files overwritten with random data before removal. Auto-purge for expired data (DPDP compliance).

🛡️

PHI/PII Masking in Logs

Aadhaar, PAN, phone numbers, emails, and policy numbers automatically masked in all log output. No PHI in plaintext logs.
